Getting Started with AndRPi

Welcome to the AndRPi documentation. Here you'll find everything you need to know about setting up and using your custom Android tablet.

Initial Setup

  1. Power On

    Press and hold the power button for 3 seconds to turn on your AndRPi tablet.

  2. Initial Configuration

    Follow the on-screen instructions to set up Wi-Fi, Google account, and basic preferences.

  3. System Update

    Check for and install any available system updates to ensure optimal performance.

Customization

Hardware Upgrades

If you purchased the BYO version, follow these steps to install your components:

  1. Remove the back cover using the provided tool
  2. Locate the RAM and storage slots
  3. Install your components following the marked orientation
  4. Replace the back cover

Troubleshooting

Common Issues

Tablet won't turn on
Ensure the battery is charged and try holding the power button for 10 seconds.
Slow performance
Check running apps and close unused ones. Consider performing a system update.
Wi-Fi connectivity issues
Toggle Wi-Fi off and on, or restart the tablet if problems persist.

Frequently Asked Questions

Can I upgrade the RAM later?

Yes, the RAM is upgradeable on all AndRPi models. Follow the hardware upgrade guide in the customization section.

What Android version is installed?

All AndRPi tablets come with the latest stable Android version optimized for Raspberry Pi.

Is the warranty void if I upgrade components?

No, hardware upgrades don't void the warranty as long as they're performed according to our guidelines.
